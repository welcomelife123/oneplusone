Countdown timer

Start: Input: Initialize countdownInterval as a variable Read the user input for endDateStr (end date and time) from the input with id "endDateInput" Start Countdown: Clear any existing countdownInterval using clearInterval Convert endDateStr to a Date object (endDate) If endDate is not a valid date: Display an alert for invalid date format Exit the function Retrieve the countdownElement with id "countdown" Define the updateCountdown function: Calculate timeRemaining as the difference between endDate and the current date If timeRemaining is less than or equal to 0: Display "Expired" in the countdownElement Else: Calculate days, hours, minutes, and seconds from timeRemaining Display the countdown in the format "days hours minutes seconds" in the countdownElement Call updateCountdown once to display the initial countdown Set an interval (countdownInterval) to call updateCountdown every second (1000 milliseconds) Reset Countdown: Clear the countdownInterval using clearInterval Clear the content of the countdownElement Clear the value of the input with id "endDateInput" End
